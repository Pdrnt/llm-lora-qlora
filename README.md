# Laboratório 07 - Especialização de LLMs com LoRA e QLoRA

Projeto da disciplina com implementação de geração de dataset sintético, fine-tuning com LoRA/QLoRA e entrega versionada no GitHub.

## Estrutura inicial
- `data/`: arquivos `.jsonl` de treino e teste
- `scripts/`: scripts de geração de dados, treinamento e inferência
- `outputs/`: saídas do treinamento e adaptadores salvos

## Ambiente
Este projeto utiliza Python com ambiente virtual (`venv`) e dependências listadas em `requirements.txt`.
